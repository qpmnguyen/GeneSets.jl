This package implements functionalities for gene set testing in Julia. Currently in development.  

Supporting pre-loaded gene sets  

- [ ] MSigDB
- [ ] GO  
- [ ] Reactome  
- [ ] KEGG  
- [ ] DAVID  

Supporting methods  
- [ ] GSEA
- [ ] Category t-test
- [ ] ROAST/CAMERA linear models
- [ ] PCGSE
- [ ] PLAGE
- [ ] GSVA
- [ ] sigPathway
- [ ] GSA (Tibshirani and Hastie)
- [ ] PCGSEA
